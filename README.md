# Gamma Page Prompt Generator

A free, no-login tool that helps Malaysian small business owners 
generate ready-to-paste prompts for Gamma AI and ChatGPT — 
so they can build a professional landing page without staring 
at a blank screen.

Built by Synthesist Transformation (ST) as a free lead tool 
for the Wizard's Handbook Series and Gamma AI classes.

## What it does
- 3-step form: business basics → page details → brand & tone
- Generates a ChatGPT brief (content planning prompt)
- Generates a Gamma AI prompt (ready to paste and build)
- Copy to clipboard, no sign-up, works offline

## How to use
1. Download or clone this repo
2. Open ST-Gamma-Prompt-Generator.html in any browser
3. Or deploy to Netlify / GitHub Pages by pointing to the HTML file
